TÍTULO DO PROJETO: Calculadora Gráfica e Cientifica, modo de Console e Janela.

OBJETIVO DO PROJETO: Este projeto tem como objetivo implementar uma calculadora com duas interfaces de usuário: modo gráfico (usando janelas) e modo texto (utilizando o console). 

A calculadora realiza operações básicas como soma, subtração, multiplicação e divisão, além de operações matemáticas avançadas como fatorial, seno e raiz quadrada.

VERSÃO ou DATA: 1.0 - 17/03/2025

AUTORES: Giulia M., Pedro H., Sthefany V., Tulio G.

INSTRUÇÕES PARA O USUÁRIO: 
Soma: Use o operador +.
Subtração: Use o operador -.
Multiplicação: Use o operador *.
Divisão: Use o operador /.
Fatorial: Digite o valor desejado como operador 1 e use o operador ! em seguida.
Seno: Digite o valor desejado como operador1 e use o operador sen (em graus) em seguida.
Raiz Quadrada: Digite o valor desejado como operador1 e use o operador sqrt em seguida.

MODO CONSOLE
	Executar o main e digitar <"texto"> como parâmetro.
	
*Caso queira executar o método String calcular, digitar operandos e operador como string. Caso use fatorial, seno ou raiz, o operando2 deve ser "0".

MODO GRÁFICO
	Executar o main e apertar "OK" em seguida. O programa foi padronizado para saída em modo gráfico.

PARA SAIR DO PROGRAMA
	Ao finalizar as operações desejadas, digitar "sair" como operando1.
